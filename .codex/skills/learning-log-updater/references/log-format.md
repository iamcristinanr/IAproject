# AI Learning Log Format

Use this structure when creating or appending to IAproject's `AI_LEARNING_LOG.md`.

```markdown
# AI Learning Log

This file tracks AI-assisted development concepts learned while building IAproject.

## YYYY-MM-DD - Milestone Title

### Concepts Learned

- Concept name: short explanation.

### Practical Actions

- Action completed in the project.

### Key Vocabulary

- Term: short definition.

### Next Concepts To Learn

- Suggested next topic.
```

Append new milestones at the top, below the introduction, so the newest learning stays visible.
